# Loan Approval Prediction using Machine Learning

This project predicts whether a loan application will be approved or rejected using Machine Learning classification algorithms.

## Project Overview

The project involves:
- Data preprocessing
- Handling missing values
- Feature scaling
- Training multiple machine learning models
- Evaluating model performance using classification metrics

## Algorithms Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Dataset

Dataset sourced using Kaggle API:
- Loan Prediction Problem Dataset

## Machine Learning Workflow

1. Import Libraries
2. Load Dataset using Kaggle API
3. Data Preprocessing
4. Feature Encoding
5. Feature Scaling
6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Visualization and Analysis

## Evaluation Metrics

The models were evaluated using:
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results

| Model | Accuracy |
|-------|----------|
| Logistic Regression | 81.3% |
| Decision Tree | 70.7% |
| Random Forest | 81.3% |

Logistic Regression and Random Forest achieved the best performance for loan approval prediction.
